# Speech to Text - AI Распознавание Речи

Веб-приложение для распознавания речи из аудиофайлов с использованием искусственного интеллекта.

## 🎯 Возможности

- **Распознавание речи** - преобразование аудиофайлов в текст с высокой точностью
- **Поддержка форматов** - WAV, MP3, M4A, FLAC, OGG, WebM
- **Большие файлы** - обработка файлов до 100MB через WebSocket
- **Прогресс в реальном времени** - отображение прогресса обработки
- **Скачивание результата** - экспорт транскрипции в текстовый файл
- **Копирование в буфер** - быстрое копирование результата

## 🛠 Технологии

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Socket.io
- **AI**: Z.ai Speech Recognition API

## 🚀 Запуск

### Установка зависимостей

```bash
bun install
cd mini-services/asr-service && bun install
```

### Запуск development сервера

```bash
# Основное приложение (порт 3000)
bun run dev

# WebSocket сервис (порт 3003)
cd mini-services/asr-service && bun run dev
```

## 📖 API

### REST API

**POST** `/api/transcribe`

Загрузка аудиофайла для транскрипции (до 10MB).

```bash
curl -X POST -F "audio=@audio.wav" http://localhost:3000/api/transcribe
```

### WebSocket

Подключение к `ws://localhost:3003` для обработки больших файлов.

**События:**
- `start-transcription` - начать транскрипцию
- `progress` - прогресс обработки
- `completed` - завершено
- `error` - ошибка

## 📝 Использование

1. Откройте приложение в браузере
2. Загрузите аудиофайл (перетаскиванием или через диалог)
3. Нажмите "Распознать речь"
4. Дождитесь завершения обработки
5. Скопируйте или скачайте результат

## 🔧 Улучшения качества

Для повышения точности распознавания:
- Используйте аудио высокого качества (16kHz+)
- Минимизируйте фоновый шум
- Используйте четкую и разборчивую речь
